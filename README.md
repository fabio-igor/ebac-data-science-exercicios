# Exercícios - Curso de Ciência de Dados (EBAC)

Repositório com os exercícios práticos desenvolvidos ao longo do curso
de Ciência de Dados da EBAC, organizados por módulo.

## Conteúdo

- **Pré-processamento de dados**: limpeza, tratamento de nulos, outliers
- **Análise exploratória**: visualização, correlação, insights de negócio
- **Modelos de Machine Learning**: Regressão Linear/Logística, Naive Bayes,
  Árvore de Decisão, Random Forest, XGBoost, SVM, K-Means
- **Estatística aplicada**: testes de hipótese, validação cruzada
- **Otimização de modelos**: GridSearchCV, RandomizedSearchCV, SMOTE

## Tecnologias

Python, Pandas, NumPy, Scikit-learn, XGBoost, Imbalanced-learn,
Matplotlib, Seaborn, SciPy
